# Supply Chain & Inventory Management System

## Overview

This project is a menu-driven Inventory Management System developed in Python using Object-Oriented Programming (OOP) concepts. It helps manage product inventory by allowing users to add new products, restock existing products, process sales, and view the inventory dashboard.

This project was developed as part of the Summer Internship Program (SIP) by Rooman Technology in Data Analytics with Python.

## Features

* View Inventory Dashboard
* Add New Products
* Restock Existing Products
* Process Customer Sales
* Automatic Stock Update
* Sales Bill Generation
* Menu-Driven Console Interface

## Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
* Dictionary
* Classes & Objects

## OOP Concepts Implemented

* Encapsulation
* Composition (SystemManager has Product objects)
* Constructor (**init**)
* Method Implementation
* Object Interaction
* **str**() Method

## Project Structure

* Product Class

  * Stores Product ID
  * Product Name
  * Price
  * Quantity
  * Restock Product
  * Sell Product

* SystemManager Class

  * Manages Inventory
  * Add/Restock Products
  * Process Sales
  * Display Inventory
